Hexo 編輯文章常用 Markdown 基本語法

Posted on 2020-08-05 Edited on 2022-08-26 In BLOG


架設好個人部落格後, 開始撰寫文章!
Hexo 在編輯文章時支援 Markdown 語法, 所以在這裡列出常用的語法.

哲學
Markdown 的目標是實現「易讀易寫」。 ( 參考: Markdown )

標題

使用 # 來表示等級一的標題，總共有六個等級，且 # 字符號和文字間要空一格。語法依此類推：

文字字體

類別 語法 顯示結果
粗體 **我是粗體** 我是粗體
斜體 *我是斜體* 我是斜體
粗斜體 ***我是粗斜體*** 我是粗斜體
刪除線 ~~我是刪除線~~ 我是刪除線
底線 <u>我是底線<u> 我是底線

文字顏色

Markdown 並沒有更改文字顏色的語法, 因此只能我們靠自己加入 HTML 語法來修改文字顏色, 例如 <font color=#008000>我是深綠色</font> : 我是深綠色,

引用區塊

使用 > 放在整列的開頭當作引用區塊，且 > 和文字間要空一格，語法範例如下：

東風夜放花千樹，更吹落、星如雨。寶馬雕車香滿路。鳳簫聲動，玉壺光轉，一夜魚龍舞。
蛾兒雪柳黃金縷，笑語盈盈暗香去。眾裡尋他千百度，驀然回首，那人卻在，燈火闌珊處。

同時也支持階層式

東風夜放花千樹，更吹落、星如雨。寶馬雕車香滿路。

表格

表格的基本語法如下，
第一列是表格的標題列，
第二列固定要是以 --- 做區隔，加欄位都是使用 | 來添加。
注意表格的每欄寬度會自動分配，所以可以忽略一切的空格(也就是每列的 | 沒有對齊也沒關係，且加多少空格也不會影響每欄的寬度)。

而每欄可以決定要如何對齊，對齊方式取決於第二列的 ---，再加上 : 符號：

  • :--- 靠左對齊。
  • ---: 靠右對齊。
  • :---: 置中對齊。
