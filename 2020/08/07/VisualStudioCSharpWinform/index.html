Winform 是微軟 .NET 開發框架的圖形用戶介面的一部分,
雖然已經推出許久的時間, 但仍然很有它應用的場景與價值.

1. 安裝整合開發環境 (IDE):

下載 Visual studio 並安裝, 因為商用問題這裡採用 Express 2017 for Windows Desktop.

  • Mircosoft 提供 Professional 和 Community 版本 (Express 版於 2017 後不再釋出)
  • 參考: Visual Studio Community 與 Express 商業免費使用之謎
